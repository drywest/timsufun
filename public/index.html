YouTube Chat Overlay (InnerTube) — Generate Overlay Link

Paste a YouTube Channel ID (eg. UC_x5XG1OV2P6uZZ5FSM9Ttw) — this will produce a permanent overlay link you can embed.

How to host: host this whole repo's frontend on GitHub Pages (static). The overlay page will connect to your Node backend server (needs to be running on a public host) — see README instructions below.
